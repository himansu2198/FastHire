🚀 FastHire

A full-stack Job Portal platform that connects job seekers with employers through a modern, scalable, and role-based system.

Built with real-world hiring workflows, focusing on clean UI/UX, modular architecture, and dynamic dashboards.

🧠 Project Idea

Traditional job platforms are cluttered and inefficient.

👉 This platform improves the experience by providing:

🎯 Clean job discovery
⚡ Fast applications
📊 Smart dashboards
🔐 Secure authentication
📈 Real-time updates

⚡ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/your-username/fasthire.git
cd fasthire
2️⃣ Frontend Setup
cd frontend
npm install
npm run dev
3️⃣ Backend Setup
cd backend
npm install
npm run server
🔐 Environment Variables

Create .env inside backend:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
